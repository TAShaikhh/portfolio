A modern, minimalist, and responsive portfolio website built with Next.js, featuring a sleek dark theme and optimized for performance.

🚀 Features
- 🌙 Dark theme optimized for eye comfort
- 📱 Fully responsive design
- ⚡ Built with Next.js 15 for optimal performance
- 🎨 Styled with Tailwind CSS and shadcn/ui components
- ✨ Smooth animations with Framer Motion
- 📧 Contact form with EmailJS integration
- 🎯 Type-safe development with TypeScript
- 🔍 SEO optimized

🛠️ Tech Stack
- Framework: Next.js 15
- Styling: Tailwind CSS
- UI Components: shadcn/ui
- Animation: Framer Motion
- Form Handling: React Hook Form + Zod
- Email Service: EmailJS
- Type Safety: TypeScript

✨ Animations and Motion
The website uses Framer Motion for animations, which may not perform optimally on older devices or phones.
